Project Goal:
To detect objects from LiDAR point cloud data by grouping nearby points into clusters and visualizing detected objects in RViz using ROS2.

Dataset Used:
lexus3-2024-04-05-gyor.mcap
Sensor: Ouster OS LiDAR
ROS2 Version: Jazzy

Current Functionality:

Raw LiDAR point cloud is received from ROS2 bag playback

PointCloud2 messages are processed in a Python ROS2 node

DBSCAN algorithm is used to group nearby points into clusters

Centroid of each cluster is calculated

Cluster centroids are published as MarkerArray messages

RViz displays both raw LiDAR points and detected cluster centers
